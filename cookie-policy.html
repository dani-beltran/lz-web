2. What Are Cookies?

2.1 Definition

Cookies are small text files that are placed on your device (computer, smartphone, tablet) when you visit a website. They are widely used to make websites work more efficiently and provide information to website owners.

2.2 Types of Cookies

Cookies can be classified in several ways:

By Ownership:

  • First-party cookies: Set by Layer Z directly
  • Third-party cookies: Set by external services we use

By Duration:

  • Session cookies: Deleted when you close your browser
  • Persistent cookies: Remain on your device for a set period

By Purpose:

  • Essential cookies: Necessary for basic functionality
  • Functional cookies: Remember your preferences and settings
  • Analytics cookies: Collect data on how you use our site
Example Cookie:
Name: layerz_session
Value: abc123def456
Domain: layerz.com
Expires: Session
Purpose: Maintain user authentication

9. Alternatives to Cookies

9.1 Similar Technologies

Besides cookies, we may use other tracking technologies:

Local Storage

  • Stores data locally in your browser
  • Persists until manually cleared
  • Used for application state and preferences
  • Managed through browser developer tools

Session Storage

  • Temporary storage for the current session
  • Cleared when tab is closed
  • Used for temporary data and forms
  • Enhances user experience

Web Beacons (Pixels)

  • Small transparent images
  • Track email opens and page views
  • Used in marketing communications
  • Can be blocked by image blocking

Fingerprinting

  • Identifies devices by their characteristics
  • Browser type, screen resolution, plugins
  • We do not use fingerprinting techniques
  • More difficult to control than cookies

9.2 Server-Side Analytics

We also collect some analytics data server-side:

  • IP addresses (anonymized)
  • User agent strings
  • Referrer information
  • Request logs and error tracking
